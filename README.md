# TT Challenge REST-API Service 

### Technologies:
**API:**
1. Nodejs
2. Framework: Expressjs 4.16.0

#### Endpoints
     1. http://localhost:8080/
        1.1 [GET] => Returns basic information about the API.
     
#### Run the Project
 Note: Requires node.js to be installed.
    
    cd [root directory of repository]
    npm i
    npm run
    curl localhost:8080


#### Run Tests
    
     cd [root directory of repository]
     mocha test
